Título

Primeiro passo: Entrar no site Dafont e procurar a fonte específica que o cliente deseja

Segundo passo: Baixar a fonte e descompactar o documento zipado

Terceiro passo:Copiar a pasta descompactada, criar uma pasta chamada fonte dentro da pasta do html

Quarto passo: criar um seletor personalizado abaixo da primeira linha do style dentro do html: @font-face

Quinto passo: Ao criar o seletor ele dará duas opções o font-family e o src, o src: url () dentro do parêntese colocar o nome da fonte baixada Exatamente como está no arquivo original. Exemplo: src: url('love larry.otf')

Sexto passo depois do nome da fonte dentro do url acrescentar format, o format existem os seguintes tipos: Opentype (para quando é otf), truetype (para quando é ttf), embedded-opentype, truetype-aat (Apple Advanced Typography) e o svg

Então ficará @font-face { font-family: ; src: url ('love larry.otf') format ('opentype') }

Caso a fonte tenha o arquivo otf e ttf e queira acrescentar os dois fazer da sguinte forma:
@font-face { font-family: ; src: url ('love larry.otf') format ('opentype'), url ('love larry.ttf') format ('truetype'); }

O font-family dentro do @font-face você pode colocar o primeiro nome da fonte. Exemplo font-family: 'Love';

Para colocar a origem de onde está a pasta da fonte baixada caso não pegue, utilizar dentro de src: url ('fonts/love larry.otf') pois assim você estará dizendo em qual pasta o arquivo está, lembrando que o caminho antes do nome da fonte deve ser o mesmo que você colocou na pasta das fontes
